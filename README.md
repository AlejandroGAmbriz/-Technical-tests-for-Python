# 🐍 Python Practice

This repository brings together a collection of technical exercises and programming challenges in Python, designed as a space for practice and experimentation. Here you will find challenges from different sources: exercises shared in forums, real experiences from technical interviews, and problems generated with the help of artificial intelligence.

Its purpose is to serve as support for those who wish to improve their skills, explore new ideas, or simply have fun solving small coding challenges. The intention is for this compendium to be a useful, accessible, and inspiring resource, both for personal learning and to motivate others on their programming journey.

*I hope you enjoy it and that it inspires you to keep creating and learning!*
